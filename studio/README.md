# Co-actor Studio

> Co-actor 프레임워크를 체험하는 웹 인터페이스.

## 실행

```bash
cd studio
pip install -r requirements.txt
python app.py
# → http://localhost:8200
```

## 기능

- **프로파일 갤러리** — 샘플 동료 에이전트 5종을 한눈에 보고 선택
- **대화** — 선택한 동료와 실시간 대화
- **PAR Loop 시각화** — 매 턴의 plan/audit 결과를 실시간으로 확인
- **약속 모니터** — 약속 준수 현황, 실패 층 분류, 관계 기울기 추적
